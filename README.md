# `tsnaivechain`

An experimental blockchain implementation, written in TypeScript and based on https://github.com/lhartikk/naivechain


## Installing

```
$ npm i
```

## Running

```
$ npm run compile
$ HTTP_PORT=5000 P2P_PORT=5001 npm run start
$ HTTP_PORT=5002 P2P_PORT=5003 npm run start
$ ...
```
